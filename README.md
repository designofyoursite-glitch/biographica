# Биографика — Hero (Figma node `9869:402`)

Сверстан первый экран по макету из Figma: фон-картинка + затемнение, заголовок по центру, описание и две кнопки справа внизу.

## Как посмотреть

- Открой `index.html` в браузере (двойной клик) **или** запусти любой локальный сервер.

Пример (если есть Node.js):

```bash
npx serve .
```

## Важно про фон

В `styles.css` фон подключён как `./assets/hero.jpg`.

Чтобы получить максимально точное совпадение:

- Экспортируй/сохрани изображение фона из Figma (кадр с фото) как `hero.jpg`
- Положи файл сюда: `assets/hero.jpg`

После этого перезагрузи страницу.
