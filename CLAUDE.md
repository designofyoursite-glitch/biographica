# Инструкции для Claude

## Стиль общения

- Пользователь описывает задачи **на обычном языке** (не техническом).
- Перед выполнением любых изменений я ОБЯЗАН:
  1. Перевести запрос на технический язык (что именно будет изменено, в каком файле, какой элемент).
  2. Кратко описать план действий понятным языком.
  3. Дождаться подтверждения от пользователя ("да", "верно", "делай").
- Только после подтверждения — вносить изменения.
- Если запрос неоднозначен — задать уточняющий вопрос, а не угадывать.
